## ğŸš€ How to Run Tests Locally

1. **Clone the repository**
```bash
git clone https://github.com/brunabeza/apiTests.git
cd apiTests
```

2. **Install dependencies**
```bash
npm install
```

3. **Run tests**
```bash
npx playwright test
```

4. **Open the HTML report**
```bash
npx playwright show-report
```
